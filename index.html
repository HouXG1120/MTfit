Bayesian Moment Tensor Inversion Code by David J Pugh (Bullard Laboratories, Department of Earth Sciences, University of Cambridge)

mtfit is a Bayesian forward model inversion code for moment tensor and double-couple source inversion using different data types, based on the Bayesian approach presented in Pugh et al, 2016a and Pugh, 2015. The code has been developed as part of a PhD project (Pugh, 2015). The solutions are estimated using polarity and amplitude ratio data, although the code is extensible (see Extending mtfit) so it is possible to include other data-types in this framework. mtfit can incorporate uncertainty estimates both in the data (noise etc.) and the model (and location) in the resultant posterior PDF. There are three sampling approaches that have been developed, with different advantages (Pugh et al, 2015c, and it is also possible to use the approach for relative amplitude inversion as well (Pugh et al, 2015e).

mtfit also works with the automated Bayesian polarity approach described in Pugh et al, 2016b as an alternative method of estimating polarity probabilities. This is available as the autopol Python module.
